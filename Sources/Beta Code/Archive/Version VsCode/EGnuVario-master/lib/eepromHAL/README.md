# EEPROMHAL Library

EepromHAL is an arduino abstraction library for Eeprom management 
									
Multi-platform :
									Arduino Pro-Mini
									Atmel SAM D21 - MKZERO 
									Espressif ESP32
									
## Setting

	in eepromHAL.h

									EEPROM_SIZE 					Eeprom size alocation

									EEPROMHAL_EXTENDED		Activate extended function
